# 🎮 Unity Hackathon Game

A Unity game prototype developed during a hackathon, focused on implementing core gameplay mechanics under time constraints.

## 🚀 Features
- Player movement system
- Core gameplay loop
- Basic game mechanics-movement, jumping , collection system

## 🛠️ Tech Stack
- Unity (C#)

## ⚠️ Note
This project was built rapidly during a hackathon, so the focus was on functionality over structure.

I am currently rebuilding this project with improved architecture, cleaner code, and additional features.

## 💡 Purpose
Demonstrates ability to build and ship a working prototype quickly and iterate on it.
